## 乔明轩的个人博客(mxygin.com.cn)

## 项目简介
该项目是我的个人博客的源码的更新版本

### 项目技术栈
- Pc端  : React,Ant-Design
- Pmd端 : React,Ant-Design
- 模拟数据 :  Mock.js
- 代码规范工具: Prettier
- 前端打包工具 : WebPack
- 前端测试工具 : Python
- api 接口测试工具:Python-Requests
- 使用 axios 与后端api通信
- 使用 Redux 进行状态管理
- 使用 React Router 处理路由
- 使用 Sass  预处理器编写样式
